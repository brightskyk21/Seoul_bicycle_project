# 🚲 Seoul Public Bike (따릉이) Optimization Project

> **160만 건의 시공간 데이터를 활용한 대여소 재배치 최적화 및 운영 효율화 시뮬레이션**







## 1. Executive Summary

* **배경**: 서울시 공공자전거 '따릉이'는 이용자가 꾸준히 증가하며 주요 교통수단으로 자리 잡았으나, 2021년 기준 약 **103억 원의 누적 적자**가 발생하며 운영 지속 가능성에 위기를 맞고 있습니다.

* **목적**: 대여 이력, 유동인구, 기상 데이터를 통합 분석하여 수요-공급 불균형을 진단하고, 데이터 기반의 재배치 시나리오를 통해 운영 효율을 극대화합니다.

* **성과**: 2023년 데이터를 학습한 XGBoost 모델($R^2$: 0.87)을 구축하였으며, 시뮬레이션을 통해 **136개 행정동의 공급 불균형 문제를 개선**하는 성과를 도출했습니다.

## 2. Tech Stack

## 3. Data Architecture & Engineering

### 📂 Data Integration

* **Dataset**: 서울시 공공자전거 대여이력(163만 건), 행정동별 생활인구, 종관기상관측(ASOS) 서울 데이터, 통계지역경계 등 6개 이종 데이터셋 통합.



### 🛠 Key Engineering

* **통계적 분포 기반 임퓨테이션(Imputation)**: 단순 결측치 삭제 대신 성별·연령 분포 비율에 따른 확률적 대체 방식을 채택하여 **가용 데이터를 63.5% 추가 확보**했습니다 (93.8만 건 → 153.4만 건).

* **공간 데이터 결합(Spatial Join)**: 대여소 좌표(WGS84)를 국내 좌표계(UTM-K)로 변환 후, 행정동 경계 데이터와 **Point-in-Polygon** 결합을 통해 결합 키가 없는 유동인구 데이터와의 연동을 구현했습니다.



---



## 4. Analysis & Insights (EDA)





* **수요-공급 불균형 진단**: 행정동별 일 평균 대여소당 대여 36회를 적정 기준으로 설정하여 분석한 결과, 한강 인근 지역의 **공급 부족**과 외곽 지역(내곡동 등)의 **공급 과잉** 현상을 정량적으로 확인했습니다.

* **유동인구-대여 전환율**: 유동인구와 대여량 간 상관계수 **0.5148**($p < 0.001$)로 높은 상관성을 보였으나, 실제 대여 전환율은 평균 **0.047%**에 불과하여 인프라 접근성 개선의 필요성을 도출했습니다.



---



## 5. Machine Learning & Optimization

### 🤖 Demand Prediction (XAI)

* **Modeling**: XGBoost Regressor를 활용하여 시계열 수요 예측 모델을 구축했습니다 ($R^2$: 0.87, RMSE: 8.96).

* **Interpretation**: SHAP 분석 결과, 행정동/자치구 내 일치 비율이 예측의 핵심 변수로 나타났으며, 이는 이용자들이 따릉이를 주로 **지역 내 단거리 순환(Last-mile)** 목적으로 활용함을 입증합니다.



### 📐 Reallocation Simulation

대여 효율($\alpha$)과 잠재 수요($\beta$) 가중치를 조합한 알고리즘을 설계하여 행정동별 최적 대여소 수를 산출했습니다.



* **시뮬레이션 결과**: 대여소 총수를 약 **2.5% 감축**했음에도 불구하고, 예측 이용량 감소는 **3.89%**로 최소화하여 서비스 품질 유지와 비용 절감을 동시에 달성할 수 있는 시나리오를 완성했습니다.
